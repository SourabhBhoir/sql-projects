
# 📊 Telecom Customer Churn Analysis (Excel Project)

## 📌 Project Overview

This project focuses on analyzing customer churn behavior in a telecom company using Microsoft Excel. The goal is to identify key factors that influence customers to leave the service and provide actionable insights to reduce churn.

## 🧰 Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Cleaning & Preprocessing
* Charts & Dashboards

## 📂 Dataset Features

The dataset includes the following attributes:

* Customer ID
* Gender
* Tenure
* Monthly Charges
* Contract Type
* Internet Service
* Churn Status

## 🔍 Key Analysis Performed

* Customer segmentation based on contract type
* Churn rate analysis across different demographics
* Impact of monthly charges on churn
* Tenure vs churn relationship
* Internet service type comparison

## 📈 Insights

* Customers with **month-to-month contracts** show higher churn rates
* Higher **monthly charges** tend to correlate with increased churn
* Customers with shorter **tenure** are more likely to leave
* Fiber users show different churn behavior compared to DSL users

## 📊 Dashboard

An interactive Excel dashboard was created to visualize:

* Churn distribution
* Customer segments
* Revenue impact
* Key performance indicators (KPIs)

## 🎯 Conclusion

This analysis helps telecom companies understand customer behavior and take data-driven decisions to improve retention strategies.
